**标注说明**：
- `【可输入】` — 用户可填写或选择的元素（文本框、下拉框、复选框、文件选择等）

- `【只读】` — 纯展示、不可交互的元素

**下拉框数据来源**：`[静态]` 硬编码；`[异步: GET /api/xxx]` 通过 API 动态加载。
